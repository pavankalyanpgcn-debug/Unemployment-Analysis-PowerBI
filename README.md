# 📊 Unemployment Analysis Dashboard (India)

## 📌 Project Overview

This project analyzes unemployment trends across different states, districts, and age groups in India using Power BI.

## 🎯 Objectives

* Analyze year-wise unemployment trends
* Identify high unemployment states
* Study unemployment across age groups

## 🛠 Tools Used

* Power BI
* Excel
* Power Query

## 📈 Key Insights

* Youth (18–25) show higher unemployment rates
* Urban regions have fluctuating trends
* Some states consistently show higher unemployment

## 📂 Files Included

* `unemployment_data.xlsx` → Dataset
* `Unemployment_Analysis.pbix` → Power BI Dashboard
* `dashboard.png` → Dashboard Screenshot

## 🚀 Outcome

Developed an interactive dashboard with filters and visualizations to support data-driven insights.
